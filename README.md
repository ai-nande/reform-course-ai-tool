# 提高改革攻坚能力 · 学员 AI 互动工具

干部培训研讨配套工具。学员在手机/电脑浏览器打开，选研讨题目 → 填 5 栏追问框架 → 一键生成提示词 → 复制到「元宝 / 豆包 / 通义」等大模型对话框中讨论。AI 仅作追问教练，不替学员下结论。

## 🔗 试用网页

**👉 [点击这里试用](https://ai-nande.github.io/reform-course-ai-tool/)**

（无需下载，手机浏览器直接打开即可。填写内容自动保存在本机，切到豆包/元宝再回来不丢失。）

## 使用步骤

1. 打开上面链接，选择本期研讨题目（六选一或「其他」自定义）；
2. 按 5 栏框架（现象/判断/为什么/要什么/卡点）填写你的想法；
3. 点「生成提示词复制」，粘贴到任意大模型对话框，让 AI 连环追问挖根因；
4. 后续步骤依次生成 MECE 结构化、角色沙盘反馈、共识输出模板。

## 设计要点

- 向导式分步，学员跟着一步步走，不堆砌；
- 本地缓存（localStorage），断网/切 App 内容不丢；
- 提示词内嵌「只追问、不给答案」强约束，保住讨论意义；
